# Provenance

## What this repository preserves

The Git history contains two historical commits and a set of MATLAB scripts under `b_kod/`, plus an archived file named `b_Kod[609].zip`. This documentation update does not alter the historical code or archive.

## What is not verified

- The source, licensing, consent status, and integrity of any dataset referenced by the scripts.
- The origin, labeling protocol, or class semantics of data expected below historical `a_Data/` paths.
- The exact MATLAB release, toolbox versions, execution environment, or original experiment settings.
- Any quantitative result, split methodology, validation procedure, or generalization claim.

## Responsible reuse

Do not add data, model files, screenshots, or external materials unless their rights and provenance are documented. Any future reproduction should use a rights-cleared, non-clinical fixture or a separately governed dataset and should document its setup independently.
